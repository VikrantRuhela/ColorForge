# 🔥 ColorForge

**Modern color palette generator for designers. Generate complementary, analogous, triadic, and monochromatic color schemes instantly

## Features

- 🎨 Pick any base color via color picker or HEX input
- ⚡ Generate **Complementary**, **Analogous**, **Triadic**, and **Monochromatic** palettes
- 📋 Copy-to-clipboard for every HEX and RGB value
- 📥 Export the full palette as a **PNG** image (via `html-to-image`)
- 🌑 Premium dark glassmorphism design with animated background orbs
- 📱 Fully responsive layout

## Tech Stack

- **React 18** + **Vite 5**
- **TinyColor2** – color math & palette generation
- **html-to-image** – PNG export
- **Inter** + **Space Grotesk** – Google Fonts

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
